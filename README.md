## Marimo 起動

```
uv run marimo --development-mode edit --headless --host 0.0.0.0 --sandbox --no-token
```

Marimo をリモートマシンで起動している場合,
表示された IP アドレスをリモートマシンの IP アドレスに変更してブラウザからアクセスする.
